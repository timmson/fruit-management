Custom issue tracker

## Requirements

### As "classis"
    * Apache >= 2.2
    * PHP >= 5.3
    * MySQL >= 5

### As Docker container
    * Docker Compose

## Installation

### Get last sources
```
git clone https://github.com/timmson/fruit-management.git 
```

### Run
```
cd fruit-management/
docker-compose run -d
```
