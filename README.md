# Fruit Management
Custom issue tracker

## Requirements

### As "classis"
    * Apache >= 2.2
    * PHP >= 5.3
    * MySQL >= 5

### As Docker container
    * Ansible
    * Docker
    * pip, docker-py

## Installation

### Get last sources
```
git clone https://github.com/timmson/fruit-management.git
```

### Startup

```
docker run -d -p 80:80 -p 3306:3306 tutum/lamp
```
