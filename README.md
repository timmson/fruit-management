[![Fruit Management ](http://img.youtube.com/vi/LGGKk6gk4lM/0.jpg)](https://www.youtube.com/watch?v=LGGKk6gk4lM "Fruit Management ")

## Requirements

### As "classis"
    * Apache >= 2.2
    * PHP >= 5.3
    * MySQL >= 5

### As Docker container
    * Docker Compose

## Installation

### Get last sources
```
git clone https://github.com/timmson/fruit-management.git 
```

### Run
```
cd fruit-management/
docker-compose run -d
```
